4  еҚЎж–№жЈҖйӘҢ

4.1 дёҚеҗҢзұ»еһӢеҚЎж–№жЈҖйӘҢзҡ„йҖүжӢ©

иҜҫжң¬дёӯе…ідәҺеӣӣж јиЎЁиө„ж–ҷзҡ„еҚЎж–№жЈҖйӘҢзҡ„ж–№жі•йҖүжӢ©д»ҘеҸҠRxCиЎЁиө„ж–ҷзҡ„жЈҖйӘҢж–№жі•йҖүжӢ©еҒҡдәҶйқһеёёеҘҪзҡ„жҖ»з»“пјҢеңЁиҝҷйҮҢдёҖе№¶е’ҢеӨ§е®¶еҲҶдә«дёҖдёӢпјҡ

еӣӣж јиЎЁиө„ж–ҷзҡ„ж–№жі•йҖүжӢ©пјҡ

  • еҪ“ n(ж ·жң¬йҮҸ)вүҘ40 дё”жүҖжңүзҡ„T(жңҹжңӣйў‘ж•°)вүҘ5ж—¶пјҢз”ЁПҮ2жЈҖйӘҢзҡ„еҹәжң¬е…¬ејҸжҲ–еӣӣж јиЎЁиө„ж–ҷд№ӢПҮ2жЈҖйӘҢзҡ„дё“з”Ёе…¬ејҸпјӣеҪ“P вүҲ Оұж—¶пјҢж”№з”Ёеӣӣж јиЎЁиө„ж–ҷзҡ„ Fisher зЎ®еҲҮжҰӮзҺҮжі•пјӣ
  • еҪ“ nвүҘ40 дҪҶжңү 1вүӨT<5 ж—¶пјҢз”Ёеӣӣж јиЎЁиө„ж–ҷПҮ2жЈҖйӘҢзҡ„ж ЎжӯЈе…¬ејҸпјҢжҲ–ж”№з”Ёеӣӣж јиЎЁиө„ж–ҷзҡ„ Fisher зЎ®еҲҮжҰӮзҺҮжі•гҖӮ
  • еҪ“ n<40пјҢжҲ– T<1ж—¶пјҢз”Ёеӣӣж јиЎЁиө„ж–ҷзҡ„ Fisher зЎ®еҲҮжҰӮзҺҮжі•гҖӮ

RГ—CиЎЁиө„ж–ҷзҡ„еҲҶзұ»еҸҠе…¶жЈҖйӘҢж–№жі•зҡ„йҖүжӢ©пјҡ

RГ—CиЎЁиө„ж–ҷеҸҜд»ҘеҲҶдёәеҸҢеҗ‘ж— еәҸгҖҒеҚ•еҗ‘жңүеәҸгҖҒеҸҢеҗ‘жңүеәҸеұһжҖ§зӣёеҗҢе’ҢеҸҢеҗ‘жңүеәҸеұһжҖ§дёҚеҗҢ4зұ»гҖӮ

  1. еҸҢеҗ‘ж— еәҸRГ—CиЎЁиө„ж–ҷ RГ—CиЎЁиө„ж–ҷдёӯдёӨдёӘеҲҶзұ»еҸҳйҮҸзҡҶдёәж— еәҸеҲҶзұ»еҸҳйҮҸеҜ№дәҺиҜҘзұ»иө„ж–ҷпјҢиӢҘз ”з©¶зӣ®зҡ„дёәеӨҡдёӘж ·жң¬зҺҮпјҲжҲ–жһ„жҲҗжҜ”пјүзҡ„жҜ”иҫғпјҢеҸҜз”ЁиЎҢГ—еҲ—иЎЁиө„ж–ҷзҡ„ПҮ2жЈҖйӘҢпјҡиӢҘз ”з©¶зӣ®зҡ„дёәеҲҶжһҗдёӨдёӘеҲҶзұ»еҸҳйҮҸд№Ӣй—ҙжңүж— е…іиҒ”жҖ§д»ҘеҸҠе…ізі»зҡ„еҜҶеҲҮзЁӢеәҰж—¶пјҢеҸҜз”ЁиЎҢГ—еҲ—иЎЁиө„ж–ҷзҡ„ПҮ2жЈҖйӘҢд»ҘеҸҠPearsonеҲ—иҒ”зі»ж•°иҝӣиЎҢеҲҶжһҗгҖӮ

  2. еҚ•еҗ‘жңүеәҸRГ—CиЎЁиө„ж–ҷ жңүдёӨз§ҚеҪўејҸгҖӮдёҖз§ҚжҳҜRГ—CиЎЁиө„ж–ҷдёӯзҡ„еҲҶз»„еҸҳйҮҸпјҲеҰӮе№ҙйҫ„пјүжҳҜжңүеәҸзҡ„пјҢиҖҢжҢҮж ҮеҸҳйҮҸпјҲеҰӮдј жҹ“з—…зҡ„зұ»еһӢпјүжҳҜж— еәҸзҡ„гҖӮе…¶з ”з©¶зӣ®зҡ„йҖҡеёёжҳҜеҲҶжһҗдёҚеҗҢе№ҙйҫ„з»„еҗ„з§Қдј жҹ“з—…зҡ„жһ„жҲҗжғ…еҶөпјҢжӯӨз§ҚеҚ•еҗ‘жңүеәҸRГ—CиЎЁиө„ж–ҷеҸҜз”ЁиЎҢГ—еҲ—иЎЁиө„ж–ҷзҡ„ПҮ2жЈҖйӘҢиҝӣиЎҢеҲҶжһҗгҖӮеҸҰдёҖз§Қжғ…еҶөжҳҜRГ—CиЎЁиө„ж–ҷдёӯзҡ„еҲҶз»„еҸҳйҮҸ(еҰӮз–—жі•)дёәж— еәҸзҡ„пјҢиҖҢжҢҮж ҮеҸҳйҮҸпјҲеҰӮз–—ж•ҲжҢүзӯүзә§еҲҶз»„пјүжҳҜжңүеәҸзҡ„гҖӮе…¶з ”з©¶зӣ®зҡ„дёәжҜ”иҫғдёҚеҗҢз–—жі•зҡ„з–—ж•ҲпјҢжӯӨз§ҚеҚ•еҗ‘жңүеәҸRГ—CиЎЁиө„ж–ҷе®ңз”Ёз§©иҪ¬жҚўзҡ„йқһеҸӮж•°жЈҖйӘҢиҝӣиЎҢеҲҶжһҗгҖӮ

  3. еҸҢеҗ‘жңүеәҸеұһжҖ§зӣёеҗҢзҡ„RГ—CиЎЁиө„ж–ҷ RГ—CиЎЁиө„ж–ҷдёӯзҡ„дёӨдёӘеҲҶзұ»еҸҳйҮҸзҡҶдёәжңүеәҸдё”еұһжҖ§зӣёеҗҢгҖӮе®һйҷ…дёҠжҳҜй…ҚеҜ№еӣӣж јиЎЁиө„ж–ҷзҡ„жү©еұ•пјҢеҚіж°ҙе№іж•°вүҘ3зҡ„й…ҚдјҚиө„ж–ҷпјҢеҰӮз”ЁдёӨз§ҚжЈҖжөӢж–№жі•еҗҢж—¶еҜ№еҗҢдёҖжү№ж ·е“Ғзҡ„жөӢе®ҡз»“жһңгҖӮе…¶з ”з©¶зӣ®зҡ„йҖҡеёёжҳҜеҲҶжһҗдёӨз§ҚжЈҖжөӢж–№жі•зҡ„дёҖиҮҙжҖ§пјҢжӯӨж—¶е®ңз”ЁдёҖиҮҙжҖ§жЈҖйӘҢжҲ–з§°KappaжЈҖйӘҢпјӣд№ҹеҸҜз”Ёзү№ж®ҠжЁЎеһӢеҲҶжһҗж–№жі•пјҲеҸҜз”ЁSASиҪҜд»¶пјүгҖӮ

  4. еҸҢеҗ‘жңүеәҸеұһжҖ§дёҚеҗҢзҡ„RГ—CиЎЁиө„ж–ҷ RГ—CиЎЁиө„ж–ҷдёӯдёӨдёӘеҲҶзұ»еҸҳйҮҸзҡҶдёәжңүеәҸзҡ„пјҢдҪҶеұһжҖ§дёҚеҗҢгҖӮеҜ№дәҺиҜҘзұ»иө„ж–ҷпјҢиӢҘз ”з©¶зӣ®зҡ„дёәеҲҶжһҗдёҚеҗҢе№ҙйҫ„з»„жӮЈиҖ…з–—ж•Ҳд№Ӣй—ҙжңүж— е·®еҲ«ж—¶пјҢеҸҜжҠҠе®ғи§ҶдёәеҚ•еҗ‘жңүеәҸRГ—CиЎЁиө„ж–ҷпјҢйҖүз”Ёз§©иҪ¬жҚўзҡ„йқһеҸӮж•°жЈҖйӘҢпјӣиӢҘз ”з©¶зӣ®зҡ„дёәеҲҶжһҗдёӨдёӘжңүеәҸеҲҶзұ»еҸҳйҮҸй—ҙжҳҜеҗҰеӯҳеңЁзӣёе…іе…ізі»пјҢе®ңз”Ёзӯүзә§зӣёе…іеҲҶжһҗпјҡиӢҘз ”з©¶зӣ®зҡ„дёәеҲҶжһҗдёӨдёӘжңүеәҸеҲҶзұ»еҸҳйҮҸй—ҙжҳҜеҗҰеӯҳеңЁзәҝжҖ§еҸҳеҢ–и¶ӢеҠҝпјҢе®ңз”ЁеүҚиҝ°зҡ„еҸҢеҗ‘жңүеәҸеҲҶз»„иө„ж–ҷзҡ„зәҝжҖ§и¶ӢеҠҝжЈҖйӘҢ(test for linear trend)гҖӮ

жңүдёҖдёӘеҸ«Mantel-HaenszelеҚЎж–№з»ҹи®ЎйҮҸзҡ„ж–№жі•жҳҜз”ЁжқҘжЈҖйӘҢдёӨдёӘжңүеәҸеҲҶзұ»еҸҳйҮҸжҳҜеҗҰеӯҳеңЁзәҝжҖ§зӣёе…ізҡ„пјҢCochran-Mantel-HaenszelеҚЎж–№з»ҹи®ЎйҮҸжҳҜеңЁе…¶еҹәзЎҖдёҠжҸҗеҮәзҡ„пјҢз”ЁдәҺй«ҳз»ҙеҲ—иҒ”иЎЁзҡ„еҲҶжһҗпјҢеҚіжҺ§еҲ¶дәҶжҹҗдёҖдёӘжҲ–еҮ дёӘж··жқӮеӣ зҙ пјҲеҲҶеұӮеҸҳйҮҸпјүд№ӢеҗҺпјҢжЈҖйӘҢдәҢз»ҙеҲ—иҒ”иЎЁзҡ„иЎҢеҸҳйҮҸе’ҢеҲ—еҸҳйҮҸжҳҜеҗҰеӯҳеңЁз»ҹи®ЎеӯҰе…іиҒ”гҖӮCochran-Mantel-HaenszelжЈҖйӘҢеұһдәҺеҲҶеұӮеҚЎж–№жЈҖйӘҢж–№жі•гҖӮ

woolf <- function(x) {
  x <- x + 1 / 2
  k <- dim(x)[3]
  or <- apply(x, 3, function(x) (x[1,1]*x[2,2])/(x[1,2]*x[2,1]))
  w <-  apply(x, 3, function(x) 1 / sum(1 / x))
  1 - pchisq(sum(w * (log(or) - weighted.mean(log(or), w)) ^ 2), k - 1)
}
